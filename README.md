# rag-suggests-career-paths
Build a model that takes in a resume and suggests career paths based on the strengths and interests. Uses an LLM from Together AI or Open Router and demonstrates how to parse PDFs and chain multiple calls together, using streamlit or Gradio for the interface part. Model should be deployed.
